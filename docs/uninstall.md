# Uninstall

## Remove the app

1. Quit KeepMirror from the menu bar.
2. Move `KeepMirror.app` to Trash.

## Optional cleanup

If you want to remove local preferences too:

1. Delete preferences file:
   - `~/Library/Preferences/com.adhamhaithameid.keepmirror.plist`
2. Remove saved photos folder if you created one specifically for KeepMirror.
3. Empty Trash.

If Start at Login was enabled, disable it in-app before uninstalling when possible.
