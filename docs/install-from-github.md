# Install From GitHub

## 1. Download

Download the latest `KeepAwake` release from GitHub.

## 2. Move To Applications

Drag `KeepAwake.app` into your Applications folder.

## 3. Open The App

Open `KeepAwake.app` once. The app runs as a menu bar utility, so look for the coffee cup icon in the menu bar.

## 4. Start A Session

- Left-click the icon to start the saved default duration.
- Right-click the icon to choose a different duration, open Settings, or quit.

## 5. Optional Setup

Open **Settings** if you want to:

- start at login
- activate on launch
- stop automatically on low battery
- stop automatically in Low Power Mode
- allow the display to sleep while the Mac stays awake

## Unsigned App Warning

If macOS shows an unsigned-app warning:

1. Right-click `KeepAwake.app`
2. Choose **Open**
3. Confirm **Open**
